## 安装与运行

### 环境要求
- Python 3.11 或更高版本
- 终端宽度 ≥ 80 列（推荐 120 列）

### 安装步骤

```bash
# 1. 进入项目目录
cd /ubuntu/doudizhu

# 2. 创建虚拟环境
python3 -m venv .venv

# 3. 激活虚拟环境
source .venv/bin/activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 运行
python main.py
```

### 命令行参数

```bash
python main.py --mcts-sims 200     # AI模拟200次（默认100，越大越强但越慢）
python main.py --seed 42           # 固定随机种子（用于复现对局）
```

## 游戏规则回顾

### 基本规则
- **54 张牌**：52 张普通牌 + 小王 + 大王
- **3 人游戏**：你 + 2 个 AI 对手
- **牌力排序**（从大到小）：大王 > 小王 > 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3

### 叫地主
- 三人轮流叫分：不叫(0分)、1分、2分、3分
- 叫 3 分者直接成为地主
- 分数最高者成为地主（获得 3 张底牌）
- 三人都「不叫」则重新发牌

### 出牌规则
- 地主先出牌
- 每手牌必须「压」过上一手（同牌型 + 更大点数），或出炸弹/火箭
- 连续 2 人过牌后，最后一个出牌者获得自由出牌权
- 先出完手牌的一方获胜

### 胜负
- 地主先出完 → 地主胜
- 任一农民先出完 → 农民胜

## 操作指南

### 游戏流程

```
1. 启动游戏 → 按 Enter 开始
2. 叫地主阶段 → 选择 0(不叫) / 1(1分) / 2(2分) / 3(3分)
3. 出牌阶段 → 选牌 → 出牌 / 过牌
4. 游戏结束 → 显示结果 → 按 Y 再来一局
```

### 选牌出牌

```
你的手牌:
    1         2         3         4
┌────┐    ┌────┐    ┌────┐    ┌────┐
│♠ A │    │♥ K │    │♣ Q │    │♦ J │   ...
└────┘    └────┘    └────┘    └────┘

> [选牌] 1-17 (可多选 如 1 3 5 或 1-5) [出牌] Enter [过牌] P [退出] Q
```

| 操作 | 按键 | 说明 |
|------|------|------|
| 选牌 | `1`-`17` | 输入序号选牌。支持多选：空格分隔 (`1 3 5`)、逗号分隔 (`1,3,5`)、范围 (`1-5`) |
| 取消选牌 | 再次输入已选序号 | 已选中的牌再次输入即取消 |
| 出牌 | `Enter` | 确认出已选中的牌 |
| 过牌 | `P` | 当前有牌需要接时才能过牌 |
| 退出 | `Q` | 放弃当前游戏 |

### 实时反馈
- 选牌时会实时显示所选牌的牌型（单张/对子/顺子/…）
- 无效牌型显示为红色 ❌
- 选中牌高亮为黄色底色
- AI 出牌有短暂延迟（模拟思考）

## AI 难度说明

### AI-1：蒙特卡洛搜索 (MCTS)

| 属性 | 值 |
|------|-----|
| 策略 | Flat Monte Carlo，每种出牌模拟 100 局 |
| 强度 | ≈ 中级玩家 |
| 特点 | 能评估大局，会保留炸弹，有农民协作意识 |
| 延迟 | 每手牌约 0.3-0.5 秒 |

### AI-2：规则 AI

| 属性 | 值 |
|------|-----|
| 策略 | 启发式规则（优先出小牌，保留大牌） |
| 强度 | ≈ 新手+ |
| 特点 | 出牌快，策略稳定可预测 |
| 延迟 | 无 |

### 调整难度

在 `src/ui/tui.py` 第 318 行附近可以修改 AI 配置：

```python
mcts_ai = MCTSAI(num_simulations=100, seed=42)  # 改 num_simulations 调整难度
```

- `50` - 简单
- `100` - 普通（默认）
- `200` - 困难
- `500` - 大师（每手牌约 2-3 秒延迟）

或通过命令行：
```bash
python main.py --mcts-sims 200
```

## 常见问题

### Q: 为什么我过不了牌？
A: 只有当前有牌需要「接」时才能过牌。自由出牌（新回合）时不能过牌——必须出牌。

### Q: 我选的牌显示「无效牌型」怎么办？
A: 检查牌型是否符合规则。例如混搭不同数量的牌（3+1 不能叫顺子）、顺子含 2 等。

### Q: AI 太强/太弱了？
A: 用 `python main.py --mcts-sims 50` 降低难度，或 `--mcts-sims 200` 提高难度。

### Q: 终端显示乱码？
A: 确保终端支持 Unicode 和 256 色。推荐使用 iTerm2、Windows Terminal、或 Kitty。

### Q: 如何运行测试？
```bash
cd /ubuntu/doudizhu
source .venv/bin/activate
python -m pytest tests/ -v
```
